# 🏃‍♂️ Exercise Tracker API

A full-stack Exercise Tracker built using Node.js, Express, and MongoDB. This project allows users to create accounts and log their exercises with details such as description, duration, and date. It also supports retrieving complete exercise history with optional filtering.

---

## 🚀 Project Overview

This application is designed to help users track their fitness activities in a simple and efficient way. Each user can add multiple exercise records, and the system maintains a complete log of all activities associated with that user.

The project demonstrates backend development concepts including REST API creation, database integration, and data handling using MongoDB and Mongoose.

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- dotenv
- CORS
